Context Menu

A Context Menu (also called as pop-up menu) is a menu that appears upon user interaction, such as a right-click. Now HTML5 allows us to customize this menu. Following are some examples that demonstrate on how to use this context menu's. Some examples do include nested menu's.

Examples

  1. At any point you can share the page on Twitter and facebook using the share menu from ContextMenu
  2. You can change the font size of this text by right clicking it and choosing the right option from context menu
  3. Change Images using ContextMenu

Explanation

contextmenu is assigned to elements. The default action of context menu depends upon whether the element or one of its ancestors has a contextmenu assigned. The default action is set by the user agernt (browser). To use context menu, we assign the contextmenu attribute to an element and a value is given. This value has to be an id of a menu element in the DOM.

  1. To have share in the menu item golbally we have assigned <body> with contextmenu="share". So our next element id DOM is a menu item with the same id as that of context menu of body
  2. Now in the menu element, we use this as type="context". This tells the user agent that we are accessing the contextmenu. An id is set.
  3. Inside menu we have <menuitem> with labels and an onclick handler. The label is what appears in the contextmenu upon the right click. And the onclink tells what action has to be performed once the menuitem is clicked. Generally this is a javascript function
  4. To have a nested menuitem, we declare nested menu's
